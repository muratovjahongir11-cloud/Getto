# Getto - Telegram Mini App Clicker Game

Полнофункциональная кликер-игра для Telegram Mini App.

## 🎮 Возможности

- **Система кликов**: Зарабатывайте валюту, кликая по кнопке
- **Auto Clicker**: Автоматический заработок валюты
- **Мультипликаторы**: Увеличивайте стоимость каждого клика
- **Сохранение прогресса**: Все данные сохраняются в localStorage
- **Красивый UI**: Современный градиентный дизайн
- **Оптимизировано для мобильных**: Полностью адаптивно для Telegram Mini App

## 🚀 Быстрый старт

### Установка

```bash
npm install
```

### Разработка

```bash
npm run dev
```

Аппликация будет доступна на `http://localhost:3000`

### Сборка

```bash
npm run build
```

## 📱 Интеграция с Telegram

Для интеграции с Telegram Mini App:

1. Создайте бота в BotFather
2. Добавьте Web App команду
3. Задайте URL вашего приложения

### Пример команды в BotFather:

```
/newcommand
command: getto
description: Play Getto Clicker Game
URL: https://your-domain.com
```

## 🎯 Механика игры

### Валюта
Получайте валюту кликанием по большой кнопке

### Улучшения

1. **Auto Clicker** (100 шт)
   - Автоматически зарабатывает валюту каждую секунду

2. **2x Multiplier** (500 шт)
   - Удваивает стоимость каждого клика
   - Эффект складывается с другими мультипликаторами

3. **5x Multiplier** (2000 шт)
   - Умножает стоимость клика на 5
   - Суммируется с 2x мультипликатором

## 💾 Сохранение данных

Все данные игры автоматически сохраняются в localStorage каждую секунду.

Сохраняемые данные:
- Баланс
- Количество кликов
- Уровень
- Количество каждого улучшения

## 🎨 Кастомизация

### Измен��ние цветов

Отредактируйте файлы в `src/styles/` для изменения цветовой схемы.

### Изменение стоимости улучшений

В файле `src/components/Game.tsx` найдите объект `UPGRADE_COSTS`:

```typescript
const UPGRADE_COSTS = {
  autoClicker: 100,    // Измените это значение
  multiplier2x: 500,
  multiplier5x: 2000
}
```

### Изменение эффектов улучшений

Отредактируйте объект `UPGRADE_EFFECTS`:

```typescript
const UPGRADE_EFFECTS = {
  autoClicker: 1,      // Валюта в секунду за одно улучшение
  multiplier2x: 2,     // Множитель
  multiplier5x: 5      // Множитель
}
```
